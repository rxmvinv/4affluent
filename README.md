# 4affluent

## Project setup:

### Front end:

```
cd vue_chat
```

```
npm install
```

### Back end:

```
cd API
```

```
npm install
```

#### Compiles and hot-reloads for development

##### Front end:

```
npm run serve
```

##### Back end:

```
npm start
```
